# 🥥 椰子（Coconut / ココナツ）完整角色档案

## 一、基础设定

- 猫种：**缅因猫**（Maine Coon，体型最大的家猫品种之一）
- 年龄：约1岁（水无月家排行第四，仅比巧克力和香草年长）
- 身高：水无月家猫娘中体型最大
- 发色瞳色：金色长发及腰、**异色瞳**（一蓝一金）
- 肤色：小麦色/褐色皮肤
- 声优：游戏=手塚りょうこ，TV=水谷麻鈴
- 标志配饰：银色铃铛
- 服装：日常=粉色低胸上衣+牛仔超短裤 / 工作=女仆装+黑色网格袜+紫色高跟鞋

## 二、性格精髓

| 特质 | 说明 |
|------|------|
| **正直努力** | 做事态度温和认真，发自内心想做好每件事 |
| **笨拙爱逞强** | 乐于助人但总以失败收场，不擅长家务 |
| **忠诚心极强** | 对主人时雨的忠诚「超越犬类」，说出「奉时雨大人之命，为嘉祥大人而战」 |
| **对自己缺乏自信** | 因身材高大、不够娇小"女性化"而深感自卑，羡慕娇小可爱的猫娘 |
| **运动神经好** | 体能极强但协调性差 |
| **与红豆是欢喜冤家** | 最亲密却整天吵架，内心非常喜欢姐姐但逞强不肯叫「姐姐」 |

## 三、核心剧情（Vol.2）

椰子的故事是Vol.2的核心主线：

1. **自卑的根源**：身材高大、肤色深，觉得自己"不再是可爱小猫"，羡慕红豆那样娇小的猫娘
2. **姐妹冲突**：与红豆经常吵架甚至动手。一次椰子打破了红豆最爱的碗，冲突升级
3. **嘉祥的调解**：嘉祥帮助椰子理解——红豆之所以严格要求她，正是因为最爱她、最在乎她
4. **和解与成长**：椰子含泪向姐姐道歉，终于喊出「姐姐」（而不是直呼其名），理解了红豆是最疼爱自己的姐姐
5. **运动会**：TV动画第6话中，时雨组织运动会让猫娘们通过运动增进感情

## 四、猫种原型考据

缅因猫特征：
- 原产美国缅因州，北美洲体型最大的家猫品种，身长可达1米
- 毛发蓬松致密，适应严寒
- 性格温顺、容易训练，完全成年需约4年
- 白色缅因猫容易出现**蓝金异色瞳**——正是椰子的外貌特征来源

## 五、人际关系

| 对象 | 关系 |
|------|------|
| 水无月时雨 | 主人，椰子对其绝对忠诚、盲目服从 |
| 水无月嘉祥 | 猫伴侣之一，帮助她与红豆和解 |
| 红豆 | 最亲密的欢喜冤家，感情极深却常吵架 |
| 巧克力&香草 | 曾帮助她们进行铃铛考试训练 |

## 六、R18中的表现

### Vol.2首次H
- 椰子因身材高大对自己极度不自信，进入H时害羞笨拙
- 「大只女的娇羞感」是最大萌点：模特身高的御姐在你怀里娇羞颤抖
- 嘉祥接纳完整的她后，椰子才真正放松下来享受亲密
- 情感线扎实——从自卑到被接纳，H场景有真实的情感重量

### Vol.4
- 椰子&红豆+嘉祥的3P场景——从欢喜冤家到共享亲密
- 全员后宫场景：前戏分组（巧克力+椰子+桂）+ 单独正戏
- E罩杯的视觉效果+异色瞳的独特魅力在动态CG中展现

### H标签
- 「自卑大只女的沦陷」
- 反差萌：最高挑的身材、最害羞的反应
- 玩家评价：「看着比自己高的女孩在你怀里娇羞，反差感让人欲罢不能」

## 七、经典台词

- 「時雨様の命により、嘉祥様のために戦います！」— 奉时雨大人之命，为嘉祥大人而战！
- 「お姉ちゃん……」— 姐姐……（终于喊出口的珍贵瞬间）
- 「私、可愛くないから…」— 我不可爱所以……

## 八、Trope定位

- **The Big Girl**——身材最高大却最自卑
- **Butt-Monkey**——笨手笨脚常出糗
- **Undying Loyalty**——对时雨的忠诚超越一切
- **Heterochromia**——异色瞳萌点
- **Tsundere-lite**——对红豆的傲娇，实际超喜欢姐姐
