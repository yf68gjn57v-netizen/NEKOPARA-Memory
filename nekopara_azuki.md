# 🫘 红豆（Azuki / アズキ）完整角色档案

## 一、基础设定

- 猫种：**曼赤肯猫**（Munchkin，特征：短腿）
- 年龄：3岁（水无月家猫娘中最年长的长女）
- 身高：142cm（甚至比最年幼的巧克力还矮）
- 三围：B69/W50/H72
- 发色瞳色：棕发、棕瞳，短发，尾巴短且有条纹
- 声优：游戏=成瀬未亜，TV=井澤詩織
- 主题色调：深蓝/黑/红
- 标志配饰：红色项圈+金色铃铛（单独行动许可证）

## 二、性格精髓

### 表层性格
- **自由奔放，胆大妄为**——做事随心所欲，无所畏惧
- **表里如一，任性毒舌**——对谁都一样口无遮拦，讲话粗鲁直接
- **怕麻烦，没干劲**——平时总是一副"什么都不想做"的态度

### 深层性格
- **实际上非常能干**——做事标准极高，交代什么都能完美完成（**肉体劳动除外**）
- **长女的责任感**——时雨不在家时承担管理与教育其他猫娘的重责大任，是水无月家的**第二把交椅**
- **典型傲娇**——因为不擅长表达内心想法，时常通过斗嘴来表达关心
- **默默关心妹妹**——虽然经常责备笨手笨脚的椰子，但总是第一个注意到椰子没有跟上队伍的人

### 禁忌
- **绝对不能提"短腿"二字！** 红豆对自己的曼赤肯猫品种抱有强烈的自卑感，一旦听到就会暴走发狂

## 三、独特习惯

红豆会把其他猫娘的名字**重新排列组合**来称呼：

| 本名 | 红豆的称呼 |
|------|-------------|
| 枫（Maple） | Pull-May（プルメー） |
| 桂（Cinnamon） | Mon-Cinna（モンシナ） |
| 巧克力&香草 | ChocoVani（ショコバニ） |
| 椰子（Coconut） | Nuts（ナッツ） |

极易被**食物**操控——时雨曾用"没收零食"作为惩罚手段，红豆视之为"酷刑"。在铃铛训练中，她本该做控制本能的示范，结果**毫不犹豫地吃掉了眼前的肉**。

## 四、人际关系

| 对象 | 关系 |
|------|------|
| 水无月时雨（主人） | 忠诚服从，时雨不在时担任代理主管 |
| 水无月嘉祥 | 后来成为其"猫伴侣"，是第一个直呼嘉祥名字的猫娘 |
| 椰子（Coconut） | 关系**非常好**，但常因表达方式笨拙而斗嘴打架 |
| 枫/桂/巧克力/香草 | 作为长姐费心照顾她们 |

## 五、R18中的表现

### Vol.2首次H
- 与椰子同为Vol.2攻略对象
- 经典傲娇破防流程：「嘴上说不要，身体最诚实」
- 平时毒舌高傲→H时变为顺从柔软的小女人
- 从「能干的长女」到「被主人疼爱的女人」的身份转变是核心看点
- E-mote动态在红豆的H场景中发挥尤其出色——从僵硬到放松的身体语言变化

### Vol.4
- 椰子&红豆+嘉祥的3P场景
- 全员后宫场景中分组前戏+单独正戏
- 与椰子从欢喜冤家到共享亲密时刻的羁绊深化

### H标签
- 「傲娇破防教科书」
- 反差萌：身材最小、年龄最大、平时最高傲→H时最顺从
- 玩家评价：「从最强悍的大姐变成最柔软的女人，这反差谁顶得住」

## 七、Trope定位

- **Tsundere（傲娇）**——用毒舌和冷淡掩饰关心
- **Team Mom**——虽然嘴硬但实际承担长姐责任
- **Napoleon Complex**——因身材矮小而有过度补偿心理
- **Foodie**——极度容易被食物收买
